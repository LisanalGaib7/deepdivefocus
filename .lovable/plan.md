## 목표
Silver Sardine과 Clownfish 두 종만 시범적으로 16×16 → 32×32로 업그레이드. 기존 실루엣/색감/포즈는 그대로 두고, 늘어난 해상도만큼 디테일(비늘 하이라이트, 지느러미 결, 눈 입체감 등)만 살짝 보강.

## 작업 범위
- 대상 파일: `src/components/common/PixelCreature.tsx`의 `sardine()`, `clownfish()` 함수만 수정
- 각 함수에 `size: 32` 추가 + 32×32 grid 교체
- `colorMap`은 기존 색상 유지하되, 음영/하이라이트용 1~2색 추가
- `glowColor`, 애니메이션 매핑, `getCreatureData` 분기 등은 변경 없음
- 다른 14종(기존) + 신규 5종(barreleye 제외)은 손대지 않음

## 방식 (Sardine)
- 기존 16×16 그리드를 2배 업스케일 후, 각 픽셀을 다듬는 방식으로 32×32 캔버스에 재배치
- 추가 디테일:
  - 측선(lateral line) 1px 라인 1줄
  - 등 쪽 음영(어두운 회색) 한 톤
  - 배 쪽 하이라이트(밝은 화이트-블루)
  - 꼬리지느러미 결 2~3px
  - 눈 동공 1px + 하이라이트 1px

## 방식 (Clownfish)
- 동일한 업스케일 → 디테일 보강
- 추가 디테일:
  - 흰 줄무늬 3개의 검은 테두리(기존엔 없음) 1px
  - 등/배 지느러미 가장자리 어두운 오렌지 라인
  - 눈에 하이라이트 1px
  - 꼬리 결 1~2px

## 검증
- `/dev/creatures` 비교 뷰어에서 32×32 토글로 신규 그리드 확인
- Bestiary(80px 셀)에서 기존 16×16 다른 종들과 어색하지 않은지 비교
- 마음에 들지 않으면 채팅 히스토리의 revert 버튼으로 이 변경분만 되돌리면 됨 (롤백 가능)
