# Drift Seahorse, Nautilus, Vampire Squid 32×32 업그레이드

Pufferfish / Mantis Shrimp / Barreleye와 동일한 방식으로, 세 생물의 픽셀 그리드를 32×32로 재제작합니다. 실루엣과 색상 정체성은 유지하되, 16×16에서는 표현할 수 없던 디테일을 추가합니다. 기존 16종의 그리드는 그대로 둡니다.

## 생물별 변경 내용

**Drift Seahorse** (`PixelCreature.tsx:105`)
- 휘어진 S자 몸통, 오른쪽을 향한 주둥이, 아래쪽으로 말린 꼬리
- 추가 디테일: 등을 따라 흐르는 등지느러미, 마디진 몸통 능선, 하이라이트가 있는 작은 눈, 머리 위 왕관(coronet)
- 색상 유지: 오렌지/앰버 계열 (`#e8a030`, `#ffcc44`, `#d09020`) + 하이라이트 톤 1개 추가

**Nautilus** (`PixelCreature.tsx:461`)
- 캔버스 약 70%를 차지하는 나선 껍데기 — 로그 나선과 각 방(chamber) 경계선을 더 깔끔하게
- 껍데기 입구(왼쪽 아래)에 촉수 다발과 작은 점눈 추가
- 색상 유지: 크림색 + 적갈색 줄무늬 (`#ede0c8`, `#a83c28`, `#3a2a2a`, `#e8c0a0`)

**Vampire Squid** (`PixelCreature.tsx:578`)
- 위쪽에 종 모양 몸통(mantle), 크고 발광하는 눈
- 아래쪽으로 8개의 팔이 부채처럼 펼쳐지고, 팔 사이를 잇는 특유의 망토(webbing) 표현
- 몸통 위에 발광기관(photophore) 점들 추가
- 색상 유지: 진한 적색 + 시안 눈 (`#cc3344`, `#991122`, `#2299ff`, `#661122`, `#ff6688`, `#ffffff`)

## 기술적 세부사항

- 각 그리드는 작은 Python 스크립트로 생성 (이전 32×32 작업과 동일한 방식) 후 `src/components/common/PixelCreature.tsx`에 붙여 넣음
- 각 항목에 `size: 32` 속성 추가 (렌더러는 이미 개별 크기 지원)
- `creatures.ts`, 애니메이션, 희귀도 등 다른 파일은 수정하지 않음
- 비교 페이지(`/creature-comparison`)는 그대로 유지하여 검증용으로 사용

## 검증 방법

작업 후 Bestiary와 비교 페이지에서 세 생물이 80px 원형 프레임 안에 깔끔하게 들어가고, 한눈에 어떤 생물인지 알아볼 수 있는지 확인합니다.
