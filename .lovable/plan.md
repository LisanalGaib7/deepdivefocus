# Nautilus & Vampire Squid 리디자인 (32×32)

`src/components/common/PixelCreature.tsx`의 두 크리쳐만 수정합니다. 색상 팔레트, 사이즈, 애니메이션은 그대로 유지.

## 1. Nautilus

- 외곽 테두리(`3`) 제거 → 16×16처럼 깔끔한 실루엣
- 껍질 내부 나선 줄무늬(`3`)는 유지
- 좌하단 빈 공간에 **촉수 다발** 추가: 입구에서 4~5가닥이 아래·왼쪽으로 뻗고 끝은 `4`로 음영
- 핀홀 눈(`3`) 1개 입구 근처 유지

## 2. Vampire Squid — 시안

테두리 없이, 통통한 종 모양 망토 + 큰 동그란 눈 + 자연스럽게 흐르는 8개 팔.
`#`=mantle(`1`), `o`=웹빙/음영(`2`), `O`=눈(`3`), `*`=하이라이트(`6`), `.`=photophore(`5`)

```text
............####............
..........########..........
.........##########.........
........############........
.......####.####.####.......   ← 머리 끝 부드러움
......##OO##*###*##OO##.....
......#OOOO##.###.#OOOO#....   ← 크고 동그란 시안 눈
......##OO####.####OO##.....
.......##########.##........
........###.######.#........
.........########...........   ← mantle 하단 곡선
..........######............
.........#  ##  #...........   ← 8개 팔 시작 (좌4 / 우4)
........#  # ## #  #........
.......#  #  ##  #  #.......   ← 각자 다른 곡선
......#  #  #  #  #  #......
.....#   #  #  #  #   #.....   ← 끝으로 갈수록 가늘어짐
....o   o   o  o   o   o....   ← `2` 음영으로 테이퍼
...o    o   o  o   o    o...
....     o  o  o  o     ....
............................
```

- 8개 팔이 좌→우로 길이/곡률 조금씩 다르게 (대칭이지만 경직되지 않음)
- 팔 사이는 옅은 `2` 웹빙으로 살짝 메워 망토 느낌
- 망토 표면에 photophore 도트(`5`) 3~4개 산포
- 색 팔레트 그대로 (`1 #cc3344`, `2 #991122`, `3 #2299ff`, `5 #ff99aa`, `6 #ffffff`). `4`(테두리)는 grid에서 미사용.
